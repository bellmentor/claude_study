== Web UI 진행상황 ==
최종 수정일: 2026-06-13

참고 이미지: ui_test/매입금수집창.png, ui_test/설정창.png

════════════════════════════════════════
[기술 스택] 확정
════════════════════════════════════════

- 백엔드: FastAPI (async, WebSocket 내장)
- 프론트엔드: Jinja2 템플릿 + vanilla JS (빌드 도구 없음)
- 실시간 로그: WebSocket (/ws/logs)
- 의존성: fastapi, uvicorn, jinja2 (+ 기존 openpyxl, playwright, pandas)

════════════════════════════════════════
[파일 구조]
════════════════════════════════════════

app/
├── __init__.py
├── server.py          # FastAPI 앱 + 모든 API 엔드포인트 + WebSocket
├── templates/
│   └── index.html     # 단일 페이지 (탭 전환은 JS)
└── static/
    ├── style.css      # UI 스타일
    └── app.js         # 탭 전환, 달력, 수집 요청, WebSocket 로그

════════════════════════════════════════
[실행 방법]
════════════════════════════════════════

1. pip install fastapi uvicorn jinja2
2. python -m app.server
3. 브라우저에서 http://localhost:8000 접속

════════════════════════════════════════
[API 엔드포인트]
════════════════════════════════════════

GET  /                   : 메인 페이지
GET  /api/accounts       : 계정 목록 조회 (계정정보.xlsx)
PUT  /api/accounts       : 계정 정보 수정 → xlsx 저장
POST /api/collect        : 매입금 수집 시작 (sites[], start_date, end_date)
GET  /api/collect/status : 수집 진행상황 조회
WS   /ws/logs            : 실시간 로그 스트림

════════════════════════════════════════
[화면 구성] 2개 탭
════════════════════════════════════════

1. 매입금 수집 탭 (메인)
   - 좌측: 달력 (월/년 드롭다운, 날짜 범위 선택)
   - 중앙: 도매사이트 테이블 (체크박스/사이트명/매입금/진행상황)
   - 우측: [수집시작] 버튼
   - 하단: 실시간 로그

2. 설정 탭
   - 계정 설정 테이블 (사이트/아이디/비번) + 추가/삭제/저장
   - 하단: 로그
