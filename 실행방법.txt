== 실행 방법 ==

※ 반드시 프로젝트 루트(C:\000.work\0.git_test)에서 실행
※ venv 활성화 필수: source .venv/Scripts/activate

────────────────────────────────
[새 PC에서 git pull 후 필요 패키지 설치]
────────────────────────────────

pip install pandas openpyxl playwright lxml fastapi uvicorn

※ lxml — 베어B2B(고도몰) 엑셀이 HTML 테이블(.xls) 형식이라 pd.read_html()에 필요
※ playwright 설치 후: playwright install chromium

────────────────────────────────
[Web UI]
────────────────────────────────

python -m app.server
→ 브라우저에서 http://localhost:8000 접속

※ web_ui_실행.bat 더블클릭 시 venv 활성화 → 서버 실행 → 웨일 브라우저로 자동 접속까지 한번에 처리됨.

────────────────────────────────
[계정정보.xlsx 준비]
────────────────────────────────

계정정보.xlsx 는 git에 올리지 않으므로, 새 PC에서 pull 후 직접 만들어야 한다.

[오너클랜]
────────────────────────────────

# 전체 실행 (로그인 → 주문조회 → 엑셀 다운로드 → 매입금 합산)
python -m dome_site.ownerclan.main

# 로그인만 테스트
python -m dome_site.ownerclan.login

# 매입금 합산만 (이미 다운로드한 엑셀이 있을 때)
python -m dome_site.ownerclan.summarize

────────────────────────────────
[새 도매처 추가 시 복붙용]
────────────────────────────────

# 전체 실행
python -m dome_site.<slug>.main

# 로그인만 테스트
python -m dome_site.<slug>.login

# 매입금 합산만
python -m dome_site.<slug>.summarize
