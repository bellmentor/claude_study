4. 엑셀 다운로드 : "요청 → 생성 대기 → 다운로드" 2단계 방식
 4-1. 엑셀 버튼 : <button class="... js-excel-download"> JS 클릭 → 레이어 열림
 4-2. 양식 선택 : <select name="formSno"> (옵션은 AJAX 로 늦게 채워짐 — 대기 필요)
  → 양식명 "정산용" 선택 (사용자가 저장해 둔 양식. '선택/정산용/주문통합리스트/주문통합리스트_주문별' 중)
 4-3. 파일명 : <input name="downloadFileName"> 에 고유 파일명 입력 (예: 정산용_YYMMDD_HHMMSS)
  → 중복 파일명 덮어쓰기 확인창 회피 + 내 요청 행 식별용
 4-4. 비밀번호 (필수) : 정산용 양식은 개인정보 3개 이상 포함 → 비밀번호 설정 필수.
  <input name="password"> / <input name="rePassword"> 에 영문/숫자/특수문자 2종, 10~16자 입력.
  (미입력 시 "비밀번호를 입력해주세요" 경고 모달이 뜨며 요청이 막힘)
 4-5. 요청 : <input type="submit" value="요청"> JS 클릭
 4-6. 생성 대기 : #tblExcelRequest 목록에서 내 파일명 행이 '생성완료' 될 때까지 폴링
 4-7. 다운로드 : 해당 행의 <button class="... js-excel-request-download"> 클릭
  → "엑셀 다운로드 사유" 모달이 뜸 : 사유 셀렉트(기본옵션 '=사유 선택=')에서 첫 사유 선택 후 '확인' 클릭
  → 그제야 download 이벤트 발생.
 4-8. 파일 구조 : 비밀번호 ZIP 안에 HTML 테이블(.xls) → zipfile(pwd)로 추출 후
  pandas.read_html(header=0) 로 파싱해 진짜 xlsx 로 변환 저장.
